# ARM_University

## 1. I teach in a university / college
   * Our free educational resources support the teaching of subjects such as Embedded Systems, 
   * SoC and IoT on contemporary systems and help save time and effort.

### 1.1 Efficient Embedded Systems Design and Programming
   * Teach the design and programming of embedded systems, and how to implement them in low-level hardware using standard C and assembly language

### 1.2 Rapid Embedded Systems Design and Programming
   * Teach how to accelerate the development of embedded systems, and rapidly prototype various embedded applications.

### 1.3 Internet of Things
   * Teach IoT principles by working through a system that connects embedded sensors using commodity smartphones, 
   * developing skills such as app development and embedded system design.

### 1.4 Digital Signal Processing
   * Help learners develop their ability to design DSP systems and create commercially viable audio applications, 
   * using high-performance and energy-efficient processors.

### 1.5 Real-Time Operating Systems Design and Programming
   * Teach foundational RTOS concepts for embedded applications, including task scheduling, memory allocation and resource management.

### 1.6 Embedded Linux
   * Teach how to develop embedded Linux-based projects, including Linux kernel configuration and custom peripheral driver development, 
   * using low cost yet powerful single board computers.

### 1.7 Introduction to System on Chip
   * Teach a typical SoC development process around Arm Cortex-M processors, which are perfectly suited for IoT needs.

### 1.8 Advanced System on Chip Design
   * Take your students through a typical advanced SoC design process on an Arm Cortex-A based platform, 
   * using standard hardware description and software programming languages.

### 1.9 Introduction to Robotic Systems
   * Introduce your students to the principles behind the development of autonomous mechatronics and robotic systems, 
   * as used in manufacturing, automotive and space applications.

### 1.10 Graphics and Mobile Gaming
   * Teach the latest graphics and game design technologies on mobile devices, with a focus on optimizing performance and reducing power consumption.

### 1.11 Introduction to Computer Architecture
   * Teach fundamental concepts of computer architecture, including how these concepts are applied and implemented in modern processors.

### 1.12 VLSI Fundamentals: A Practical Approach
   * Teach the fundamentals of VLSI design, including how the theories and concepts can be applied in the design of simple logic circuits 
   * and in the physical implementation of a simplified microprocessor.
